Introduction

This web application allows users to manage albums from the Chinook database. You can view, add, edit, and delete albums easily with a clean and simple interface.

Features

  • View Albums: Browse all albums and their artists.
  • Add Album: Create a new album by selecting an artist and providing a title.
  • Edit Album: Modify existing album details.
  • Delete Album: Remove albums with confirmation to prevent accidental deletions.

How to Use

  1. View Albums: Click the View Albums link to see all albums listed in a table with options to edit or delete each entry.
  2. Add Album: Click Add Album and fill in the form. Select the artist from the dropdown and enter the album title, then submit.
  3. Edit Album: From the albums list, click the Edit button for an album, update fields, and save changes.
  4. Delete Album: Click the Delete button next to an album, confirm deletion to permanently remove the album.

Navigation

Use the navigation bar at the top of each page to quickly access Home, View Albums, and Add Album pages.

Note: Always confirm actions carefully, especially deletions, as these cannot be undone.
